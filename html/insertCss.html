외부스타일 시트

  • 많은 페이지에 동일한 스타일을 적용하려고 할 때 좋은 방법
  • html 페이지는 link 태그를 이용해서 스타일 파일 연결
  • 외부 스타일 시트는 단순한 텍스트 파일이다
  • html태그는 포함하지 않아야 한다
  • .css 라는 확장자를 가져야한다

내부스타일 시트 (internal style sheet)

  • html안에 css를 정의
  • style 태그를 정의하여 head요소 안에 넣는다
  • 외부 스타일 시트보다 우선순위가 높다

인라인 스타일 시트 (inline style sheet)

  • 각각의 요소마다 스타일을 지정
  • 이 방법을 사용하면 스타일 시트의 장점을 잃게 된다
  • 권장하지 않으므로 꼭 필요한 경우에만 사용
  • 우선 순위가 가장 높다
