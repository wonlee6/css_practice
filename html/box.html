여백을 포함하지 않은 크기로 px, %, em등의 단위로 된 수치 사용 가능

인라인 요소에는 적용 불가
  • auto : width 속성은 화면 전체를 차지하며, height속성은 콘텐츠 높이에 따라 자동 조절
  • length: 길이 단위 값으로 적용되는 방식은 box-sizing 속성 값에 따라 다름 ehight속성의 경우 콘텐츠 높이가 지정된 값보다 크더라도 요소의 높이는 변경되자 않음 음수를 지정할 수 없음
  • percentage : 자신을 포함한 요소의 값을 참조하여 비율로 결정 heigth 속성의 경우 length처럼 콘텐츠 높이로 계산된 값이 지정된 값보다 크더라도 요소의 높이는 변경되지 않으며 음수 지정 불가
